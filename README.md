- run this command in your terminal with whatever IP address corresponds to your car
  ```bash
  export car_ip_address=192.168.0.XXX
  ```
- run **./run.sh** to run the version of experiment.py on the car
- run **./upload.sh** to replace the version of experiment.py on the car with the local version
- run **./both.sh** to upload a new experiment.py and run it
- The password for all commands is: exec
